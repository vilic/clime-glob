# Clime Glob Objects

Provides easy access to glob matching for Clime command.

## Installation

```sh
npm install clime-glob --save
```

## Classes

| Constructor | Description |
| --- | --- |
| `Matches` | Find file names that match the given comma separated glob patterns. |
